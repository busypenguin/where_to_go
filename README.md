# Куда пойти — Москва глазами Артёма

Готовый сервис сайта о самых интересных местах в Москве. Авторский проект Артёма.

## Запуск

Для запуска сайта вам понадобится Python третьей версии.

Скачайте код с GitHub. Установите зависимости:

```sh
pip install -r requirements.txt
```

Создайте базу данных:

```sh
python3 manage.py migrate
```

Запустите разработческий сервер:

```sh
python3 manage.py runserver
```

Для загрузки новых мест используется команда `load_place` Дя запуска нужно ввести в терминал:

```sh
python3 manage.py load_places http://адрес/файла.json
```

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл `.env` в каталоге `where_to_go/` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.


Доступны 3 переменные:
- `DEBUG` — дебаг-режим. Поставьте `True`, чтобы увидеть отладочную информацию в случае ошибки.
- `SECRET_KEY` — секретный ключ проекта
- `ALLOWED_HOSTS` — см [документацию Django](https://docs.djangoproject.com/en/3.1/ref/settings/#allowed-hosts)

```sh
SECRET_KEY=h9cte7...h$6yte5...Fhchs9y
```
Данные выше приведены для примера. Создайте свой `SECRET_KEY`
